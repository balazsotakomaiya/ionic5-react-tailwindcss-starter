## Installation
Ensure that you have Node, NPM & the [Ionic CLI](https://ionicframework.com/docs/intro/cli) installed.

```shell
npm install
```

## Development

Run the app in the browser
```shell
npm run start
```

Run the app on Android
```shell
npm run build
ionic capacitor add android
npm run dev:android
```

Run the app on iOS
```shell
npm run build
ionic capacitor add ios
npm run dev:ios
```
